# 💻 Programação 2 - Aula 08: Prática 04

## 📋 Sobre os Exercícios

A lista foca na aplicação de conceitos fundamentais de Java, como estruturas de decisão, laços de repetição e manipulação de arrays e strings. Abaixo estão os desafios propostos:

- [ ] **Exercício 01:** Programa que recebe a idade de um usuário e verifica se o voto é obrigatório, facultativo ou não permitido.
- [ ] **Exercício 02:** Sistema de classificação de nadadores baseado na idade (Infantil A/B, Juvenil A/B e Adulto).
- [ ] **Exercício 03:** Calculadora de valor a ser pago em um posto de combustíveis, aplicando uma tabela de descontos progressivos para Álcool e Gasolina.
- [ ] **Exercício 04:** Loop de contagem regressiva de 10 a 0 finalizando com a mensagem "Lançar!".
- [ ] **Exercício 05:** Algoritmo que calcula a média do array `int[] notas = {7,8,6,10,9}` e informa quantas notas ficaram acima da média.
- [ ] **Exercício 06:** Iteração sobre o array `String[] nomes = {"Ana","Bruno","Alê","Aline","Clara"}` utilizando `foreach` para imprimir apenas os nomes que começam com a letra 'A'.
- [ ] **Exercício 07:** Manipulação de String para receber uma palavra e imprimi-la de trás para frente (ex.: "casa" → "asac").
- [ ] **Exercício 08:** Gerador de tabuada de multiplicação (1 a 10) de um número inserido pelo usuário.
- [ ] **Exercício 09:** Validador de senha usando laço `while`, que continua solicitando a entrada até que a senha correta ("1234") seja digitada para liberar o "Acesso permitido".
