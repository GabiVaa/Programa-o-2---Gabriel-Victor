# 📚 Atividade 04 - Estruturas de Repetição e Arrays

---

## 📝 Descrição dos Exercícios

A atividade é composta por 9 exercícios práticos, abrangendo desde validações simples até manipulação de vetores:

* [cite_start]**`Exercicio1.java`**: Recebe a idade do usuário e determina se ele possui obrigatoriedade de voto[cite: 8].
* [cite_start]**`Exercicio2.java`**: Classifica nadadores em categorias (Infantil, Juvenil ou Adulto) de acordo com a faixa etária informada[cite: 10, 16].
* [cite_start]**`Exercicio3.java`**: Calcula o valor a ser pago em um posto de combustível, aplicando descontos diferenciados para Álcool e Gasolina com base na litragem[cite: 20, 24].
* [cite_start]**`Exercicio4.java`**: Implementa um laço de repetição para realizar uma contagem regressiva de 10 a 0, exibindo "Lançar!" ao final[cite: 28].
* [cite_start]**`Exercicio5.java`**: Processa um array de notas, calculando a média aritmética e identificando quantos alunos ficaram acima da média[cite: 30].
* [cite_start]**`Exercicio6.java`**: Utiliza a estrutura `foreach` para filtrar e imprimir nomes que iniciam com a letra 'A' em um array de Strings[cite: 31].
* [cite_start]**`Exercicio7.java`**: Recebe uma palavra e utiliza lógica de repetição para invertê-la (ex: "casa" para "asac")[cite: 32].
* [cite_start]**`Exercicio8.java`**: Gera a tabuada de multiplicação de 1 a 10 para um número qualquer inserido pelo usuário[cite: 33].
* [cite_start]**`Exercicio9.java`**: Valida o acesso através de uma senha pré-definida, utilizando um laço `while` que persiste até a entrada correta[cite: 37, 39].
