# 📚 Atividade 06 - Manipulação de Strings

Este diretório contém a resolução dos exercícios práticos da **Aula 06** da disciplina de Programação 2 da Universidade Federal Rural de Pernambuco (UFRPE). 

O foco desta atividade foi o estudo e a prática de métodos da classe `String` em Java, essenciais para a manipulação de textos.

---

## 📝 Descrição dos Exercícios

A atividade é composta por 6 exercícios práticos, cada um focando em um método específico de manipulação de Strings:

* **`Exercicio1.java`**: Utiliza o método `equals()` para comparar se dois nomes digitados pelo usuário são exatamente iguais (diferenciando maiúsculas de minúsculas).
* **`Exercicio2.java`**: Utiliza o método `equalsIgnoreCase()` para comparar duas palavras ignorando as diferenças entre letras maiúsculas e minúsculas.
* **`Exercicio3.java`**: Utiliza o método `length()` para contar e exibir a quantidade de caracteres de uma palavra informada pelo usuário.
* **`Exercicio4.java`**: Utiliza o método `substring()` para dividir a palavra "Programação" em duas partes ("Programa" e "ação").
* **`Exercicio5.java`**: Utiliza o método `charAt()` para extrair e exibir o primeiro, o terceiro e o último caractere da palavra "Computação".
* **`Exercicio6.java`**: Utiliza o método `indexOf()` para buscar uma palavra específica dentro de uma frase digitada pelo usuário e retornar a sua posição (índice) ou `-1` caso não seja encontrada.

---

## ⚙️ Como executar os códigos

1. Certifique-se de ter o [Java JDK](https://www.oracle.com/java/technologies/downloads/) instalado na sua máquina.
2. Clone o repositório ou baixe os arquivos desta pasta.
3. Abra a pasta em sua IDE favorita (IntelliJ IDEA, Eclipse, VS Code, etc.).
4. Como a maioria dos exercícios exige entrada de dados, execute o arquivo desejado e interaja com o terminal (console) da sua IDE para digitar as informações solicitadas.
