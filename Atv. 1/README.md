# 📚 Atividade 01 - Programação 2 (UFRPE)

Este repositório contém a resolução dos exercícios práticos da **Aula 03 - Prática 01** da disciplina de Programação 2 da Universidade Federal Rural de Pernambuco (UFRPE).

---

## 🚀 Tecnologias Utilizadas

* **Linguagem:** Java
* **IDE:** IntelliJ IDEA

---

## 📝 Descrição dos Exercícios

A atividade foi dividida em três partes focadas nos fundamentos da linguagem Java:

* **`Exercicio1.java`**: Prática com variáveis do tipo `String`. O código declara variáveis para nome e sobrenome e exibe os valores no terminal.
* **`Exercicio2.java`**: Trabalha com múltiplos tipos de dados (`String`, `int` e `double`). O objetivo é armazenar informações (nome, idade e altura) e exibi-las em uma única mensagem formatada.
* **`Exercicio3.java`**: Focado em operadores aritméticos. O código recebe duas variáveis numéricas (`double`) e realiza as quatro operações matemáticas básicas (soma, subtração, multiplicação e divisão), exibindo o resultado de cada uma.
