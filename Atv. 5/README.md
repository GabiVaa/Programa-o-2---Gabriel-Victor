# Exercícios Práticos de Lógica em Java ☕

Este repositório contém a resolução de quatro exercícios básicos de lógica de programação utilizando a linguagem **Java**. Cada exercício foi separado em sua própria classe para facilitar a leitura e os testes.

## 📋 Lista de Exercícios

* **Exercicio1.java (Cálculo da Hipotenusa):** Recebe os valores de dois catetos de um triângulo retângulo e retorna o valor da hipotenusa utilizando o Teorema de Pitágoras e o método `Math.sqrt()`.

* **Exercicio2.java (Maior Valor em Array):** Recebe um array de números inteiros e percorre seus elementos para identificar e retornar o maior valor presente nele.

* **Exercicio3.java (Busca de Elemento em Array):** Recebe um array de `Strings` e uma palavra de busca. Retorna `true` se a palavra existir dentro do array e `false` caso não seja encontrada.

* **Exercicio4.java (Contador de Pares e Ímpares):** Recebe um array de inteiros e retorna um novo array de duas posições: a primeira posição contém a quantidade total de números pares, e a segunda contém a quantidade de números ímpares.
