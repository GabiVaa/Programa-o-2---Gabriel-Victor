# 💻 Atividade Prática 02 - Programação 2

Repositório destinado à resolução e entrega da 2ª atividade prática da disciplina de Programação 2 da Universidade Federal Rural de Pernambuco (UFRPE).

## 📝 Sobre os Exercícios

Este projeto contém 6 algoritmos desenvolvidos em **Java**, abordando conceitos de entrada e saída de dados, operadores matemáticos e estruturas condicionais:

* **Exercício 1:** Lê um número inteiro e imprime seu dobro e triplo.
* **Exercício 2:** Avalia a idade informada e verifica a permissão para votar (16+ anos).
* **Exercício 3:** Calcula o valor de uma compra e aplica 8% de desconto para valores totais acima de R$ 100,00.
* **Exercício 4:** Calcula a estimativa de litros de combustível que podem ser comprados com um valor específico.
* **Exercício 5:** Simula a venda de ingressos de cinema, aplicando 50% de desconto para opções de meia-entrada.
* **Exercício 6:** Calcula o valor final de uma compra contendo múltiplos produtos, já incluindo a porcentagem da taxa de serviço.

## 🛠️ Tecnologias Utilizadas

* Java
* IntelliJ IDEA
